# MasterChrome Tab Dashboard 🌟

A beautiful, premium Google Chrome extension designed to organize your workspace, catch duplicate links, and clean up massive tab hoards automatically. Built with elegance and fully personalized tab management by Michelle Wang.

---

## 🚀 Installation Steps (For New Users)

Since this is a highly customized, local developer extension, you can install it directly without needing the Chrome Web Store:

### 1. Download the Code
1. Download this repository folder to your local computer and navigate into `Documents/projects/MasterChrome`.

### 2. Turn on Developer Mode in Chrome
1. Open your Google Chrome browser.
2. Type `chrome://extensions/` into the top address bar and hit **Enter**.
3. Look at the **top-right corner** of the screen and switch **Developer mode** to **ON**.

### 3. Install Your Extension
1. Click the **Load unpacked** button located in the top-left corner of the page.
2. Choose the `Documents/projects/MasterChrome` folder inside this downloaded repository.
3. **You're all set!** Open a **New Tab** (`Cmd + T` on Mac or `Ctrl + T` on Windows) to view your beautiful new workspace dashboard!

---

## ✨ Core Features
- **Automatic Domain Sorting**: Instantly groups tabs by specific tools (GitHub, YouTube, Google Workspace, Internal corporate Go Links, etc.).
- **Duplicate Cleanup Engine**: Intelligently catches redundant links hidden behind tab-suspenders and allows instant one-click cleanup.
- **Priority Display**: Automatically bubbles your top 3 most-active groups directly to the very top of the screen.
